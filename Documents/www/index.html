Overview of Valkyrie OS

The Valkyrie Operating System is an operating system designed for the x86 32-bit architecture. OS versions in the future are planned to support x64. The operating system is where all executable files are Java Archives JAR or Java Bytecode or .class files. The Valkyrie Operating System started in October 2025, originally as a 512-byte micro operating system. Since then, extensive features have been developed and integrated, making it a fully-functional OS with bootloader, kernel, and JVM support. Key Features

  • Java-Centric Design: Run JAR files and Java bytecode directly on the OS
  • x86 Architecture: Full support for 32-bit x86 systems with future x64 support planned
  • Custom Bootloader: Two-stage bootloader for flexible boot configurations
  • Kernel-Based: Full multitasking kernel with hardware abstraction layer
  • Integrated JVM: Execute Java programs natively on bare metal
  • File System Support: FAT12, FAT16, FAT32, and ext2 file systems

System Components

Stage 1 Bootloader: Initializes hardware and loads Stage 2 from disk
Stage 2 Bootloader: Sets up memory, GDT, IDT, and loads the kernel
Kernel: Manages processes, memory, I/O, and provides system calls
Java Virtual Machine: Executes Java bytecode and JAR applications

Getting Started

To build Valkyrie OS, you'll need a cross-compiler toolchain. Full build instructions are available in the Downloads section and Documentation pages. The source code is available on GitHub.
